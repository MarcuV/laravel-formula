# Laravel 
Setup Laravel development environment in few minutes.

Laravel stack includes: 
- Latest PHP7 docker image, 
- NGINX (tuned with letsencrypt and self signed certificate support), 
- PostregSQL, 
- Redis cache, 
- RabbitMQ, 
- ELK (Logs), 
- Vault (Credentials management)
